# Ubuntu 22 Jammy, Vagrant provisioner, VirtualBox machine, x64 arch.


───────────────────────────────────────────────────────────────────────
# Installation
───────────────────────────────────────────────────────────────────────
	
	install virtualbox
	install vagrant
	
───────────────────────────────────────────────────────────────────────
# Configuration
───────────────────────────────────────────────────────────────────────
	
	vagrant up

───────────────────────────────────────────────────────────────────────
# Operation
───────────────────────────────────────────────────────────────────────
	
	vagrant box list
	
	vagrant ssh
	
	
───────────────────────────────────────────────────────────────────────
# Destruction
───────────────────────────────────────────────────────────────────────
	
	vagrant box remove  
	
	vboxmanage ungregistervm vbox-ubuntu-faktory --delete-all
